# hangman-rs

Learning Rust with a little game.

```
You guessed: o
    ___
    |  |     Guesses: t a e u o
    |  O
    | / \
    |
  __|__      _ o _ _ e _
```

## Requirements

These are the versions used, but it should work on older versions as well.

- Rust 1.68.2
- Cargo 1.68.2

## How to run

```
git clone https://github.com/haroflow/hangman-rs
cd hangman-rs
cargo run
```
